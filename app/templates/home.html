CompBot, The Faculty of Science and Technology's AI Chatbot.

The CompBot project is a chatbot focused on giving students information about the faculty of science and technology. It is a great tool to use when students have questions about the faculty without the hassle of sending a plethora of emails or going in person to speak to someone from the faculty.

HOW IT WORKS

Best Features to Help You

Answer questions related to the faculty in a timely manner giving a human-like response.

Save user data so it can start chatting with users right where they left off.

Save questions / statements and learn from them so it can use them in the future.

Notify administrator for questions that cannot be answered. Admin will respond to these directly via email.
