CompBot is a deep learning chatbot that uses python, tensorflow and natural language processing(NLP), Natural Language Understanding (NLU), and Machine Learning (ML) to imitate human-like interactions with a user. The chatbot we design will be used for the specific purpose of answering questions about the Faculty of Science and Technology.

The Code Behind CompBot

Architecture

  • The user sends the chatbot a question and that is sent to the business layer for processing
  • The NLP/NLU function then translate the user's question into a data that the model can understand.
  • That data is then sent to the decision engine to find patterns and heuristics based on the training data that the model was trained on.
  • The decision engine then uses the patterns to quickly identify a suitable answer which is then drawn from the data source.
  • The data connector then sends the answer to the NGL function which then decodes the data in natural language.
  • The answer is then brought to the backend connector and then to the user.
Need to Know

Frequently Asked Questions.

How does CompBot work?

CompBot is a chatbot that uses deep learning neural networks to learn from the data that it is trained on. CompBot outputs an appropriate response to the user based on the question asked.

What is the purpose of CompBot?

The purpose of CompBot is to answer questions about the Faculty of Science and Technology. It will not be able to provide tailored responses to students as it is a general purpose chatbot.

How long does it take to respond?

CompBot should answer questions within 3 seconds of the question being asked.
